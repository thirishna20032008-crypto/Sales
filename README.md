Sales Prediction Using Machine Learning

Project Overview

This project uses Machine Learning to predict sales based on advertising data. The model analyzes advertising spending and estimates future sales values.

Objective

The main objective of this project is to build a prediction model that can estimate sales using input advertising data.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

Dataset

The dataset contains advertising information and sales values used to train and test the model.

Steps Performed

1. Imported required libraries
2. Loaded the dataset
3. Performed data preprocessing
4. Split data into training and testing sets
5. Trained the Machine Learning model
6. Predicted future sales values
7. Evaluated the model performance

Prediction Output

Predicted Sales:

- 15.96
- 10.80

Interpretation:
The model successfully predicted future sales values using advertising data.

Conclusion

This project demonstrates how Machine Learning can be used for sales forecasting and decision-making based on advertising data.
